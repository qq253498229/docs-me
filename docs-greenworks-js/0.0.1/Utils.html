工具方法

1. 移动文件夹

greenworks.Utils.move(source_dir, target_dir, [success_callback], [error_callback])

  • source_dir String

  • target_dir String

  • success_callback Function()

  • error_callback Function(err)

将 source_dir 源目录移动到 target_dir 目标目录。

2. 创建压缩包

greenworks.Utils.createArchive(zip_file_path, source_dir, password, compress_level, success_callback, [error_callback])

  • zip_file_path String

  • source_dir String

  • password String: 字符串表示没有密码

  • compress_level Integer: 压缩系数0-9，数值越大压缩率越大

  • success_callback Function()

  • error_callback Function(err)

将 source_dir 源目录打包成一个ZIP压缩包文件。

3. 解压压缩包

greenworks.Utils.extractArchive(zip_file_path, extract_dir, password, success_callback, [error_callback])

  • zip_file_path String

  • extract_dir String

  • password String: 空字符串表示没有密码

  • success_callback Function()

  • error_callback Function(err)

将压缩文件 zip_file_path 解压缩到指定的 extract_dir 目录。
