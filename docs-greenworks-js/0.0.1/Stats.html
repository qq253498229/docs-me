统计

1. 获取用户的统计值

greenworks.getStatInt(name)

  • name String: 用户统计的API 名称

返回一个 Integer 值，表示用户的统计状态值。

2. 获取用户的float格式统计值

greenworks.getStatFloat(name)

  • name String: 用户统计的API 名称

返回一个 Float 值，表示用户的统计值。

3. 设置/更新指定统计值

greenworks.setStat(name, value)

  • name String: 用户统计的API 名称

  • value Integer or Float: 用户的统计值

返回一个 Boolean 值，表示方法调用是否成功。

4. 持久保存统计值

greenworks.storeStats(success_callback, [error_callback])

  • success_callback Function(game_id)

    • game_id: 这些统计数据用于哪个游戏

  • error_callback Function(err)

将变动的统计与成就数据发送至服务器进行持久保存。

5. 重置所有统计状态

greenworks.resetAllStats(isResetAchievement)

  • isResetAchievement Boolean: 是否重置成就

重置当前用户的统计状态。
