线程转储 (threaddump)

threaddump 端点提供应用程序 JVM 的线程转储。

获取 JSON 格式的线程转储

要以 JSON 格式获取线程转储，可以按照以下基于 curl 的示例，使用适当的 Accept 头发送 GET 请求到 /actuator/threaddump：

curl 'http://localhost:8080/actuator/threaddump' -i -X GET \
-H 'Accept: application/json'
