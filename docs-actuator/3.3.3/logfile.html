日志文件 (logfile)

logfile 端点提供对应用程序日志文件内容的访问。

检索日志文件

要检索日志文件，请发送 GET 请求到 /actuator/logfile，如下基于 curl 的示例所示：

curl 'http://localhost:8080/actuator/logfile' -i -X GET

检索日志文件的一部分

使用 Jersey 时不支持检索日志文件的一部分。

要检索日志文件的一部分，请使用 Range 头发送 GET 请求到 /actuator/logfile，如下基于 curl 的示例所示：

curl 'http://localhost:8080/actuator/logfile' -i -X GET \
-H 'Range: bytes=0-1023'

上述示例检索了日志文件的前 1024 字节。得到的响应类似于以下内容：

HTTP/1.1 206 Partial Content
Accept-Ranges: bytes
Content-Type: text/plain;charset=UTF-8
Content-Range: bytes 0-1023/4707
Content-Length: 1024
