功法搭配基本原则

先选主力摧破技能，也就是输出技能，以大拙手为例 。build都是围绕着主力摧破功法来搭配

  • 看好属性：金刚，那就把内功调整到100金刚

  • 武学：拳掌，说明是用拳套做武器的

  • 蓄势消耗：4崩，那就挑个崩多的拳套武器

  • 功法属性：力道，说明伤害根据力道计算，所以要尽量高的堆力道

  • 发挥需求：不够就用宝物凑，最大210%

  • 选择门派功法只观察：内功、身法、绝技，也就是前三行

  • 每种类型只看自己级别的，比如2精纯的时候就只看789三品，不要好高骛远

  • 观察每种功法的类型、运功效果、正练逆练，是不是适合自己的摧破
