发展阶段(7品)

  • 买神一品的马，方便跑图（砸钱）

  • 买高级诗书技艺书用来参考（砸钱）

  • 买佛学技艺书(催迫需要佛学)和7品宝物

  • 买7品防具和武器，淬毒

  • 邀请几个同道，比如百花空桑的医术和毒术（弥补自己的不足）

  • 买加突破功法成功率的建筑的心材并建造（路过公输坊顺便买)

    • 买到书后暂时不要修习，等买齐了会太吾村盖好建筑一起练了（利用好buff）

  • 买所有内功、7品身法、7品绝技（威望不够就用诗书换，砸钱）

  • 会太吾村利用练功房全给练了
