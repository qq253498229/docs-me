开始刷资源

医术

医术可以制作药物，放在村里的熟药铺卖掉，实现财富自由

  • 在回春堂买药，买下9品，中8品药材就行

    • 医术造诣低的时候可以奇和正都不选，有一半概率随机到正，这样就能越级做药

    • 医术造诣高了就选正

  • 村里盖4个满级熟药铺卖药

  • 如果医术造诣特别低，可以买5品工具能加90造诣

厨艺

厨艺可以制作食物恢复悟性，用来读书；也可以赠予NPC提高亲密刷资源，实现资源自由

  • 读上中下(789)三本厨艺书

  • 去服牛帮买食材，用上7品材料做5品菜，顺便买个好点的锅

  • 去大城市(襄阳)找下九流送菜(到亲密)，然后邀请同道，交换所有资源后派去打工

    • 中庸，仁善，刚正的下九流送一次菜到亲密；叛逆和唯我需要送两次

  • 如果厨艺造诣特别低，可以去服牛帮买4品菜(不到2000块)送
