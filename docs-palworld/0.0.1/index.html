幻兽帕鲁

三个基地分配

  • 主基地建筑，牧场，手工

  • 矿场金属，石炭(更新后可以合并到主基地)

  • 配种牧场

矿场基地坐标

  • 190，-40

育种基地坐标

  • -255，44（开阔）

  • -478，-85

  • -413，-6

  • -292，-50

  • -78，193（拉风）

20只上限，16只育种，1个发电，1个生火（做蛋糕），1个冷却保鲜（冰箱），1个空余（比如商人）

推荐帕鲁

云海鹿-跑图神器

棉悠悠+幻月蝶=云海鹿

阿努比斯-天生打工人

猫蝠怪+燧火鸟=阿努比斯

绵悠悠-提高出售价格

  • 高贵+5%交易价格

  • 贵族+3%交易价格

带5只去卖东西，就能提高40%卖出价格，以及降低40%买入价格

  • 【生火】腾炎龙、焰煌、狱焰（阎）王

  • 【挖矿】碎岩龟、焰煌、阿努比斯

  • 【浇水】覆海龙、碧海龙

  • 【手工】阿努比斯、百合女王

  • 【种植、收获、制药】百合女王

  • 【伐木】狱焰（阎）王、白绒雪怪

  • 【搬运】不限：4速通用，推荐单职搬运

  • 【发电】波鲁杰克斯、不限(基本不缺电）

  • 【冷却】唤冬曽、吹雪狐

  • 【牧场】蜂蜜-骑士蜂，牛奶一波霸牛，鸡蛋一皮皮鸡，羊毛一美露帕/棉悠悠

新手流程

  • 前期先边做任务边升级，等能育种了先做个云海鹿帮助探图，能育种基本就算渡过前期了。

  • 接下来一点点根据需要配出战斗、工作、跑图帕鲁，开分基地，刷出金色装备，攻略全部boss

  • 终端在10级时解锁第二个据点，15级解锁第三个据点。

  • 终端每升级1次，帕鲁上限加1。
