幻兽帕鲁0.4.12

人形帕鲁

  • 4只满星鲨小子就不用说了。洗点之后先点满血量再点攻击，操作好的可以先攻。

  • 然后就是属性帕鲁，选克制对应BOSS的。新版本塔主是冰系的，推荐用火系的火麒麟克制。

  • 属性帕鲁技能全选抗揍的，提高生存能力。然后升满星，帕鲁魂强化满血量和防御。

  • 然后是首饰，攻击吊坠1、2、3都带上，3种首饰可以叠加。第四个带耐热防寒的衣服。

  • 4只满星鲨小子(突袭指挥官 铁臂军师)

  • 属性帕鲁(金刚之躯 传说 顽强肉体 受虐狂)

  • 疾旋鼬-骑乘赋予龙属性

  • 桃旋鼬-骑乘赋予火属性/火麒麟/燧火鸟

  • 雷角马-骑乘赋予雷属性/迅雷鸟/雷鸣鸟

  • 邪麒麟-骑乘赋予暗属性/噬魂兽/雷冥鸟/露娜蒂

  • 碧海龙-骑乘赋予水属性

  • 唤冬兽-骑乘赋予冰属性且强化冰属性攻击

  • 唤夜兽-骑乘赋予暗属性且强化暗属性攻击

  • 踏春兔-并肩作战时赋予草属性
