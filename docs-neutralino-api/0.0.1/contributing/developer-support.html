开发者支持

如果你有关于 Neutralinojs 的问题，可以通过以下方式找到答案：

  • 在 StackOverflow 上发布问题，并附上 neutralinojs 标签。

  • 使用 Slack 和 Discord 频道。

  • 发送邮件至 neutralinojs[AT]gmail.com。

  • 在 GitHub Discussions 上创建讨论主题。

Bug 报告和功能请求

没有完美的软件，但我们可以通过努力打造优秀的软件。通过报告错误和提交功能请求，帮助我们改进 Neutralinojs。

请在相应的 GitHub 仓库中创建问题，用于功能请求和错误报告：

  • Neutralinojs 原生 API：创建问题

  • Neutralinojs JavaScript API：创建问题

  • neu CLI：创建问题

  • 启动模板：创建问题

  • 文档：创建问题
