1.14. 说明

1.14.1. 强度说明

  • S - 牛！逼！

  • A - 很强

  • B - 还行

  • C - 能玩

  • D - 垃圾

  • F - 别！碰！

  • 次要等级: +/- = 更高/更低

评分是在理想化的T3地图中，每分钟杀怪数量的数量决定的

1.14.2. 是否强化地图

  • ✔✔ = 这个构建最适合强化地图

  • ✔ = 这个构建可以使用强化地图

  • ❌= 这个构建不推荐强化地图

1.14.3. 造价

  • ✔✔✔ = 造价级低

  • ✔✔ = 造价还可以

  • ✔ = 需要一定的积累

  • ❌= 需要级好的装备或者是稀有难以掉落的装备才能玩

1.14.4. HC(专家硬核模式)

  • ✔✔ = 非常适合硬核模式

  • ✔ = 稍微改一点就可以适应硬核模式

  • ❌= 不推荐/在硬核模式中很难生存
