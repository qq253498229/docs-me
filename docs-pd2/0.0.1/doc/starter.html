开荒简略流程

1. 整体流程

  • 开荒剧情，点满主要输出技能

    • 找到适合当前职业的技能

      • 亚马逊6级用充能一击，35做出和平甲后洗点转召唤流，用女武神、诱饵输出

      • 死灵法师开局用骷髅，20级左右转成粘土石魔，石魔专精加到10点。遇到物免招火焰石魔

      • 德鲁伊加满召唤中间一列，左边狼獾1点，右边3个藤各1点

      • 法师到达A5用火系12级的技能火墙可以快速击杀8pp矫正者和督军山克升级

    • 15级左右到A2

      • 获得经验和怪物等级相关，所以人物等级不要过高或过低，详情查看 经验获取

    • 23级左右到A3

      • 级别不够去塔拉夏古墓练几级，挑精英怪打，7个古墓逛逛就到了

      • A1火焰佣兵18级开启活力光环，极大增加移动速度，开荒期间强烈推荐

      • A1冰冻佣兵24级开启回蓝光环，增加魔法恢复速度，缺蓝的职业带这个

      • A2防御佣兵6级开启反抗光环，增加防御和PDR，比较通用。其它佣兵光环请可以查看 雇佣兵光环效果

    • 快速过剧情，小怪能不打就不打，直奔A5第一个小站练级

      • 使用 .pp x 调整pp数来增加经验，根据角色强度，保证效率的前提下尽量提高

      • 普通难度练到37、38级左右就可以继续通关了

      • 必做的任务 尽量做掉，否则之后还要回头来做

    • 通关普通难度

    • 快速过剧情，除了 必做的任务 之外，小怪能不打就不打，直奔A5第一个小站练级

      • 噩梦难道58级左右就可以继续通关了

    • 通关噩梦难度

  • 噩梦MF为进军地狱做准备

    • 法师可以直奔地狱 卡位杀虫子

    • 噩梦安姐、虫子、劳模都不错

  • 地狱MF，升级到80级满足开T图要求

    • 选择合适的85场景

  • 高效MF虫子或直接开刷T图

    • 虫子使用的是巴尔的掉落，安姐使用的是劳模的掉落

    • 根据角色定位不同选择也不同。法师自带TP就多刷刷虫子；尸爆死灵建议直接刷T图

  • MF出装腐化，做RW及手工，目标核心BD。

  • 刷小黑：刷虫子出的暗金/绿装卖商店，有5%几率出小黑，出高级符文(最大28#)和手工材料

    • 如果前期打不过小黑就把暗金/绿装攒起来等以后再卖，记得用配方 钥匙+暗金/绿装 将装备转换成绿色项链节省格子

  • 刷T3+亮金+腐化地图

  • 刷3BB（大量暗金）

  • 刷6BOSS（火炬）

  • 刷BOSS：大黑（DC）毁灭小符

  • 刷BOSS：路西安对项链进行二次腐化

  • 刷最终BOSS：拉斯玛（DC腐化石）

2. 开荒期间常用的符文之语

  • 13级能用天底头(4+3)增强回蓝，可以少喝几个蓝屏，非必要；

  • 15级能用怨恨(6+1+5)3孔近战武器，通常用长柄做给米山用，100%OW，非必要；

  • 17级能用隐秘甲(7+5)增加RUN/FCR/FHR以及毒抗，非常推荐；

  • 21级能用先祖盾(8+9+7)增加巨量抗性，A5任务正好给一套符文，随便捡个3孔盾打上，推荐；

  • 25级能用精神(7+10+9+11)剑没有好用武器的话可以做一个；

  • 27级能用眼光(长柄/弓)=8+3+7+12=回蓝光环，一般给雇佣兵带

  • 27级能用知识头(9+12)，+1技能和电抗；
