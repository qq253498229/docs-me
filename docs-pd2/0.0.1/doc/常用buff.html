常用Buff整理

1. 祈祷

提供生命恢复和击中回复生命

  • A3冰冻佣兵附带光环，最高18级

2. 抵抗火焰

6级提供10%火焰伤害45%抗火

  • 暗金精英单手剑，魔灵杀手，提供6级抵抗火焰灵气

  • 暗金拓展双手剑，特迪斯法雷-芙法米，提供6级抵抗火焰灵气

  • RW3孔头盔(4+21+26)，摇曳烟火，提供1-6级抵抗火焰灵气

3. 抵抗冰冷

6级提供10%冰冷伤害45%抗冰

  • 暗金精英单手剑，冰霜之风，提供6级抵抗冰冷灵气

4. 抵抗闪电

6级提供10%闪电伤害45%抗电

  • 暗金精英单手剑，光之军刀，提供6级抵抗闪电灵气

5. 反抗

13级提供128%防御以及6%减少物理伤害；16级提供146%防御以及8%减少物理伤害；18级提供162%防御以及9%减少物理伤害

  • A2防御佣兵附带光环，最高18级

  • RW流亡13-16级

6. 净化

降低毒素和诅咒的持续时间

  • A3佣兵

  • RW瘟疫15-17级

7. 活力

提供移动速度，耐力上限和耐力恢复速度

  • A1佣兵

  • RW和谐10级

8. 冥思

提供法力回复速度

  • A1佣兵

  • RW眼光12-17级

9. 救赎

几率清理地面尸体，提供生命和法力回复

  • RW凤凰10-12级

10. 救助

增加元素伤害，提供元素(电/冰/火，没有毒)抗性

  • RW天顶8-12级

11. 力量

增加攻击力

  • A5佣兵

  • RW最后希望17级

  • 暗金圣堂武士的力量6-8级

12. 圣火

范围持续火焰伤害，击中附加火焰伤害，准确率。受协同技能加成

  • RW飞龙12级

  • 正义之手12级

  • 火焰号叫24-28级

13. 荆棘

提供伤害反弹

  • A2佣兵

  • 锐锋15级

  • 荆棘18-21级

  • 沉默23-25级

14. 祝福瞄准

增加致命攻击(DS)几率和准确率

  • A2佣兵

15. 专注

增加增加攻击力，降低攻击中断的可能(固定20%)

  • 骄傲18-20级

  • 薄雾8-12级

  • 魔仆3-6级

16. 神圣冰冻

范围持续冰冷伤害，减慢附近敌人，击中附加冰冷伤害，准确率。受协同技能加成

  • 厄运16级

  • 寒冰18级

  • 破碎之墙16级

17. 神圣冲击

范围持续闪电伤害，击中附加闪电伤害，准确率。受协同技能加成

  • A3佣兵

  • 梦境13级

  • 泽瑞尔的决心25-30级

18. 庇护所

降低敌人魔法抗性，击中附加魔法伤害，准确率。受协同技能加成

  • A4佣兵

  • 碧蓝怒火10-12级

  • 天堂装束1级

  • 庇护17-20级

19. 狂热

提供攻击力，攻击速度，准确率

  • 野兽8-10级

  • 信心12-15级

20. 审判

降低附近敌人防御力和元素抗性(没有毒)

  • 无限

  • 随机词缀

  • 暗金精英长柄武器，死神的丧钟，33%几率击中释放31级伤害加深

  • 暗金精英双手钉头槌，碎脑锤，8%几率击中释放33级伤害加深
