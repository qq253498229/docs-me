新星电法（刷地图）

  • 武器：双手杖50fcr底子做的无限85fcr，技能最好是3闪电专精+3新星

  • 副手：cta+警戒之墙

  • 手套：秘术师，腐化30fcr

  • 戒指：soj*2

  • 靴子：纱织

  • 腰带：技能腰带，腐化30fcr

  • 盔甲：亚特玛的哭喊20fcr，腐化1技能，没有就带魁黑刚，或者奥马斯40fcr

  • 头盔：格里芬25fcr，in3个电珠

  • 项链：马拉，腐化无法冰冻（全身至少有一件，或者是两个50%）

fcr200(正常只有170，腐化的不好就需要用戒指和项链凑)，fhr86

技能

  • 心灵传动max+充能弹max+新星max+电专精max

  • 能量护盾点到大概60+

属性：力量够了，剩下全加能量

  • 佣兵武器：暴风尖塔6孔eth，in13#*6

  • 佣兵盔甲：纯洁25+20+33+15，大量防御+cb+毒抗

  • 佣兵头盔：残暴33#+32#+13#，大量防御+嘲讽+ll+fhr+无法冰冻

  • 佣兵手套：卓古拉之握

  • 佣兵腰带：长传之耳

  • 佣兵靴子：蚀肉骑士
