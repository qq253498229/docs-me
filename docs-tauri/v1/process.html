进程(process)

原文地址

对当前进程执行操作。

当 tauri.conf.json 中的 build.withGlobalTauri 设置为 true 时，也可以通过 window.__TAURI__.process 访问此包。

Functions

exit

  • exit(exitCode?: number): Promise<void>

立即以给定的 exitCode 退出。

示例

import { exit } from '@tauri-apps/api/process';
await exit(1);

Since: 1.0.0

参数

名称

类型

默认值

描述

exitCode

number

0

要使用的退出代码。

返回: Promise<void>

一个表示操作成功或失败的 Promise。
