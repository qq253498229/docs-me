枪械

1. 总结：

  • 开局先用免费枪：

  • 1级猎物用【维兰特.22LR经典版】

  • 2-6级猎物用【游侠.243步枪】

  • 4-6级猎物用【ZARZA-10.308（追踪者）】

  • 7-9级猎物用【马尔默7MM马格南步枪】

然后升级

  • 11级(1.2W)/29级(2.4W)解锁测距望远镜

  • 可选：攒4W块买【F.L.Sporter.303抛光弹】针对4-6级猎物，容易补枪。蒂阿拉罗瓦国家公园完成任务送

  • 攒7.5W买【.300坎宁马格南步枪疆界款】针对7-9级猎物，要求步枪得分6000

2. 1级

  • 维兰特.22LR经典版（5星推荐，1级枪械天花板，免费）

  • 1级，2.5重量，归零范围：50-100-150m

  • 推荐1级猎物

  • 优点：

    • 1.弹容量大，射速快，容错率高；

    • 2.惊扰范围小，约150米；

    • 3.对于150米外的猎物，没打着也会“警戒中”而不会逃跑；

    • 4.在装备.22LR包壳空尖弹的情况下，对于所有弹药等级为1的猎物都是1枪秒

  • 缺点：

    • 1.弹道偏高，对于体型小的猎物需要稍微压低枪口；

    • 2.50米以内的目标建议近距离归零，否则弹道会明显偏高

3. 2级

  • 游侠.243步枪（5星推荐，2级猎物大杀器，初始赠送）

  • 2-6级，3.5重量，归零范围：75-150-300m

  • 推荐2级猎物

  • 优点：开局白送，各项数值平均，2级猎物最佳枪械

  • 缺点：

    • 伤害偏低，容易皮肉伤；

    • 3级以上的猎物比较疲软；

    • 弹道差，配合归零点技能只适合250m以内的猎物，超出距离需要抬高枪口预瞄

4. 4-6级

  • .270猎人步枪（2星推荐，赫西费尔登地图任务必备枪械，1.2W块）

  • 4-8级，4重量，归零范围：75-150-300m

  • 推荐4级猎物

  • 优点：

    • 1.入门级4-8级枪械，各方面都很均衡；

    • 2.默认归零情况下，200米内弹道特别直；

    • 3.对于0氪党在解锁7mm步枪之前是非常好的过渡武器

  • 缺点：

    • 1.重量较高；

    • 2.射程一般，不宜设计250米以外的目标；

    • 3.伤害一般，5级以上的猎物就比较难了

  • ZARZA-10.308（追踪者）（5星推荐，低后座版M1，4-8级枪械天花板，全图必备，公认神器，免费）

  • 4-8级，3.5重量，归零范围：75-150-300m

  • 推荐4-6级猎物

  • 优点：

    • 1.弹道直，没有明显便宜，激活归零点后，300米指哪打哪，默认归零状态下，200米弹道几乎无下坠；

    • 2.伤害高；

    • 3.半自动设计模式，后坐力低，控枪更容易

  • 缺点：1.后坐力依旧不小，建议把“后坐力管理”技能点满

  • F.L.Sporter.303抛光弹（5星推荐，公认神器，4W块）

  • 4-8级，3.5重量，归零范围：75-150-300m

  • 推荐4-6级猎物

  • 优点：

    • 1.射程远，弹道直，300米内精度高（配合归零技能）

    • 2.高伤害，

    • 3.低后座，拉栓快，易上手，易补枪

  • 缺点：较贵，4W块

5. 4-7级

  • 马尔默7MM马格南步枪（4星推荐，完全取代7mm摄政王/女王，免费）

  • 4-9级，3.5重量，归零范围：75-150-300m

  • 推荐4-7级猎物，前期甚至可以替代7-9级枪械

  • 优点：

    • 1.高伤害，足够应对<8级的所有猎物，

    • 2.泛用性强，兼顾4-9级猎物，前期背包容量有限的情况下，可以用来替代7-9级枪械，

    • 3.与7mm摄政王/女王相比，重量轻了1个点

  • 缺点：

    • 1.在300米归零点时，弹道会略微向右偏，需要将准星向左微调；

    • 2.拉栓速度慢，是栓动步枪中最慢的，不适合连杀和补枪；

    • 3.不宜猎取8、9级猎物，因为.7mm弹药的伤害不足以快速击杀

6. 7-9级

  • .300坎宁马格南步枪疆界款（5星推荐，公认神器，全图必备，攒钱必买，4W块）

  • 7-9级，4重量

  • 推荐7-9级猎物

  • 优点：

    • 1.射程远，弹道直，320米内精度高（配合归零技能）

    • 2.重量轻，是7-9级步枪中最轻的，

    • 3.高伤害，对付熊、牛都绰绰有余，

    • 4.弹容量大

  • 缺点：后坐力大，新手不建议用16倍镜
