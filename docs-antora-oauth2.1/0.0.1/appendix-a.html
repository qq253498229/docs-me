附录A. 增强型巴科斯范式（ABNF）语法

本节使用[RFC5234]中的符号为本规范中定义的元素提供增强型巴科斯范式（ABNF）语法描述。以下ABNF根据Unicode代码点[W3C.REC-xml-20081126]进行定义；这些字符通常采用UTF-8编码。元素按首次定义的顺序呈现。

以下部分定义中使用了来自[RFC3986]的“URI引用”定义。

以下部分定义中使用了以下常用定义：

VSCHAR     = %x20-7E
NQCHAR     = %x21 / %x23-5B / %x5D-7E
NQSCHAR    = %x20-21 / %x23-5B / %x5D-7E

A.1. "client_id" 语法

client_id 元素定义在 章节 2.4.1:

client-id     = *VSCHAR

A.2. "client_secret" 语法

client_secret 元素定义在 章节 2.4.1:

client-secret = *VSCHAR

A.3. "response_type" 语法

response_type 元素定义在 章节 4.1.1 和 章节 6.4:

response-type = response-name *( SP response-name )
response-name = 1*response-char
response-char = "_" / DIGIT / ALPHA

A.4. "scope" 语法

scope 元素定义在 章节 1.4.1:

 scope       = scope-token *( SP scope-token )
 scope-token = 1*NQCHAR

A.5. "state" 语法

state 元素定义在 章节 4.1.1, 章节 4.1.2, 和 章节 4.1.2.1:

 state      = 1*VSCHAR

A.6. "redirect_uri" 语法

redirect_uri 元素定义在 章节 4.1.1, 和 章节 4.1.3:

 redirect-uri      = URI-reference

A.7. "error" 语法

error 元素定义在 章节 4.1.2.1, 章节 3.2.4, 7.2, 和 8.5:

 error             = 1*NQSCHAR

A.8. "error_description" 语法

error_description 元素定义在 章节 4.1.2.1, 章节 3.2.4, 和 章节 5.3:

 error-description = 1*NQSCHAR

A.9. "error_uri" 语法

error_uri 元素定义在 章节 4.1.2.1, 章节 3.2.4, 和 7.2:

 error-uri         = URI-reference

A.10. "grant_type" 语法

The grant_type 元素定义在 章节 3.2.2:

 grant-type = grant-name / URI-reference
 grant-name = 1*name-char
 name-char  = "-" / "." / "_" / DIGIT / ALPHA

A.11. "code" 语法

code 元素定义在 章节 4.1.3:

 code       = 1*VSCHAR

A.12. "access_token" 语法

access_token 元素定义在 章节 3.2.3:

 access-token = 1*VSCHAR

A.13. "token_type" 语法

token_type 元素定义在 章节 3.2.3, 和 章节 6.1:

 token-type = type-name / URI-reference
 type-name  = 1*name-char
 name-char  = "-" / "." / "_" / DIGIT / ALPHA

A.14. "expires_in" 语法

expires_in 元素定义在 章节 3.2.3:

 expires-in = 1*DIGIT

A.15. "refresh_token" 语法

refresh_token 元素定义在 章节 3.2.3 和 章节 4.3:

 refresh-token = 1*VSCHAR

A.16. 端点参数语法

新端点参数的语法定义在 章节 6.2:

 param-name = 1*name-char
 name-char  = "-" / "." / "_" / DIGIT / ALPHA

A.17. "code_verifier" 语法

code_verifier的ABNF如下。

code-verifier = 43*128unreserved
unreserved = ALPHA / DIGIT / "-" / "." / "_" / "~"
ALPHA = %x41-5A / %x61-7A
DIGIT = %x30-39

A.18. "code_challenge" 语法

code_challenge的ABNF如下。

code-challenge = 43*128unreserved
unreserved = ALPHA / DIGIT / "-" / "." / "_" / "~"
ALPHA = %x41-5A / %x61-7A
DIGIT = %x30-39
