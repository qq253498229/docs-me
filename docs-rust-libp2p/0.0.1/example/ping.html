Ping示例

描述

这个 ping 示例展示了如何创建一个节点网络，这些节点建立连接，协商 ping 协议，并相互 ping。

用法

要运行这个示例，请按照以下步骤操作：

  • 1.在第一个终端窗口中，运行以下命令：

cargo run

这个命令启动一个节点，并打印 PeerId 和监听地址，例如 Listening on /ip4/0.0.0.0/tcp/24915。

  • 2.在第二个终端窗口中，用以下命令启动示例的新实例：

cargo run -- /ip4/127.0.0.1/tcp/24915

将 /ip4/127.0.0.1/tcp/24915 替换为从第一个终端窗口获得的第一个节点的监听地址。

  • 3.这两个节点将建立连接，协商 ping 协议，并开始相互 ping。

结论

ping 示例演示了如何使用 libp2p 创建一个简单的 p2p 网络并实现一个 ping 协议。通过运行多个节点并观察 ping 行为，用户可以了解 libp2p 如何促进对等体之间的通信和交互。

代码

Cargo.toml

[package]
name = "ping-example"
version = "0.1.0"
edition = "2021"
publish = false
license = "MIT"

[package.metadata.release]
release = false

[dependencies]
futures = { workspace = true }
libp2p = { path = "../../libp2p", features = ["noise", "ping", "tcp", "tokio", "yamux"] }
tokio = { workspace = true, features = ["full"] }
tracing = { workspace = true }
tracing-subscriber = { workspace = true, features = ["env-filter"] }

[lints]
workspace = true

main.rs

#![doc = include_str!("../README.md")]

use futures::prelude::*;
use libp2p::{noise, ping, swarm::SwarmEvent, tcp, yamux, Multiaddr};
use std::{error::Error, time::Duration};
use tracing_subscriber::EnvFilter;

#[tokio::main]
async fn main() -> Result<(), Box<dyn Error>> {
    let _ = tracing_subscriber::fmt()
        .with_env_filter(EnvFilter::from_default_env())
        .try_init();

    let mut swarm = libp2p::SwarmBuilder::with_new_identity()
        .with_tokio()
        .with_tcp(
            tcp::Config::default(),
            noise::Config::new,
            yamux::Config::default,
        )?
        .with_behaviour(|_| ping::Behaviour::default())?
        .with_swarm_config(|cfg| cfg.with_idle_connection_timeout(Duration::from_secs(u64::MAX)))
        .build();

    // Tell the swarm to listen on all interfaces and a random, OS-assigned
    // port.
    swarm.listen_on("/ip4/0.0.0.0/tcp/0".parse()?)?;

    // Dial the peer identified by the multi-address given as the second
    // command-line argument, if any.
    if let Some(addr) = std::env::args().nth(1) {
        let remote: Multiaddr = addr.parse()?;
        swarm.dial(remote)?;
        println!("Dialed {addr}")
    }

    loop {
        match swarm.select_next_some().await {
            SwarmEvent::NewListenAddr { address, .. } => println!("Listening on {address:?}"),
            SwarmEvent::Behaviour(event) => println!("{event:?}"),
            _ => {}
        }
    }
}
