示例代码

展示如何使用rust-libp2p的一些 示例。

  • 聊天应用程序示例: 一个基本的聊天应用程序，演示了libp2p、mDNS和gossip协议。

  • 分布式键值存储: 一个展示libp2p以及mDNS和Kademlia协议的基本键值存储。

  • 基本文件共享应用程序: 具有提供或根据名称定位和获取文件的对等方的基本文件共享应用程序。
    虽然这个示例显然展示了如何使用Kademlia和请求-响应协议构建基本的文件共享应用程序，但实际上这个示例的目标是展示如何将rust-libp2p集成到更大的应用程序中。

  • IPFS Kademlia: 演示如何在IPFS网络上执行Kademlia查询。

  • IPFS 私有实现: 使用gossipsub、ping和identify协议来实现ipfs私有蜂群功能。

  • Ping: 小型ping克隆程序，向对等方发送一个ping，期望得到一个pong作为响应。查看教程，了解逐步构建示例的指南。

  • Rendezvous: Rendezvous协议。参见规范。
